Description

El proyecto Hacienda AgroNova tiene como objetivo el diseño y desarrollo de una base de datos integral para gestionar eficientemente todas las operaciones de una finca agrícola. Esta finca produce una variedad de productos, como aguacates, maíz y leche, y requiere una solución robusta para manejar aspectos clave como la gestión de empleados, maquinaria, ventas, inventarios, y relaciones con proveedores y clientes.
